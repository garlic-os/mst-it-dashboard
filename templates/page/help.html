Netgroups Are Missing

The netgroups list is not automatically updated when new netgroups are created. To manually update the list, run the Netgroup Update Script.

Update Netgroups

Forms Are Out Of Date

The forms do not automatically update when forms are modified in the Y: Drive. To manually update the forms, run the Form Import Script.

Import Forms

Buildings Are Missing

The buildings list is manually created by us. There is no good way to automate its creation. To manually update the buildings, follow the following steps:

1. Open R:\userweb\ithelp\data\dashboard\buildings.js

2. Add ["####"], "Building Name(Street Address)"]

   #### is the four letter building code.

   Building Name is the friendly name for the building.

   Street Address is the street address of the building.

Remote Desktop Does Nothing

To use the RDP Hyperlinks on a Windows Desktop, you will need to install the rdp:// protocol. To install the protocol, run the RDP Protocol Installer.
